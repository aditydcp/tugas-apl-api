## How to Run
- install dan konfigurasi npm

- download proyek ini

- buka terminal dan pastikan directory yang terbuka adalah repository ini

- ketik command `npm start`

- buka browser dan akses `localhost:3000`

## How to Run (using VS Code)
- buka terminal di VS Code
- 
- ketik command `npm install`
- 
- ketik command `npm start`
- 
- klik tautan `localhost:xxxx`
